# DragonEgg Job Hunter

DragonEgg Job Hunter is an AI-agent workflow that helps discover AI startups and generate outreach emails.

## Features

- AI startup dataset generation
- Cold outreach email templates
- AI-assisted research workflow
- Built using OpenClaw agent

## Files

startups.md  
Dataset of AI startups.

cold_emails.md  
Cold outreach email templates.

outreach_tracker.md  
Track outreach progress.

## Goal

Help discover and connect with AI startups.
